# ✅ Проверка развёртывания

После того как запустил скрипт `auto_deploy.sh`, проверь следующее:

## 1️⃣ Проверь фронтенд

Открой в браузере:
```
https://elendes10.beget.tech/frontend/
```

Должен увидеть:
- ✓ Заголовок "📚 Moi-Kursi"
- ✓ Поле поиска сверху
- ✓ Сообщение "Выбери курс слева"
- ✓ Загрузка файлов не должна быть заблокирована

**Если не загружается:**
- Проверь что домен правильный
- Проверь HTTPS (могут быть SSL ошибки - это нормально для новых серверов)
- Перезагрузи страницу (Ctrl+F5)

## 2️⃣ Проверь API

Открой в браузере:
```
https://elendes10.beget.tech/backend/api/v1/courses
```

Должен увидеть JSON:
```json
{
  "success": true,
  "data": [],
  "message": "Success"
}
```

(Пока пустой массив, потому что нет курсов)

**Если видит ошибку 404 или 500:**
- Проверь что `.htaccess` загружен в `backend/`
- Проверь что `backend/api/index.php` существует
- Смотри логи сервера (если есть доступ)

## 3️⃣ Проверь БД (если есть доступ)

Если у тебя есть доступ к MySQL на сервере Beget:

```bash
# Подключись по SSH
ssh root@155.212.139.51

# Проверь таблицы
mysql -u moi_kursi_user -pMoiKursi2026Secure! moi_kursi_db
SHOW TABLES;
```

Должны быть 3 таблицы:
- courses
- sections
- lessons

## 4️⃣ Если файлы не загрузились

Если скрипт не смог загрузить файлы по SSH, загрузи их вручную:

1. Открой FileZilla (или другой FTP клиент)
2. Подключись к Beget (данные в панели управления)
3. Загрузи файлы:
   - `backend/` → `public_html/backend/`
   - `frontend/` → `public_html/frontend/`
   - `docs/` → `public_html/docs/`
   - `.env` → `public_html/.env`

## 5️⃣ Если БД не создалась

Запусти `install.php` вручную:

1. Убедись что `.env` загружен на сервер
2. Открой в браузере:
   ```
   https://elendes10.beget.tech/backend/install.php
   ```
3. Следуй инструкциям на странице
