# Moi-kursi: Личная платформа видеокурсов

Веб-платформа для организации и просмотра личной коллекции видеокурсов (84GB, 20+ курсов). Подобна GetCourse, но для приватного использования одного пользователя.

## 🎯 Цели

- Создать единую платформу для всех видеокурсов
- Систематизировать как на GetCourse (курсы → разделы → уроки)
- Освободить место на компьютере (переместить на облако)
- Удобный поиск и фильтрацию по курсам/темам
- Доступ с браузера и мобильных устройств

## 🏗️ Архитектура

```
Браузер (Web + Mobile)
    ↓ (API Requests)
Beget Хостинг (Blog тариф)
    ├─ PHP/Node.js приложение
    ├─ MySQL База данных
    └─ REST API
    ↓ (Video URLs)
Mail.ru Cloud (1TB)
    └─ Хранилище видеофайлов (84GB)
```

## 💰 Стоимость

- **Beget (Blog):** 520 ₽/месяц
- **Mail.ru Cloud (1TB):** 133 ₽/месяц
- **Итого:** ~650 ₽/месяц

## 🚀 Быстрый старт

### Требования
- PHP 7.4+ или Node.js 14+
- MySQL 5.7+
- Beget хостинг
- Mail.ru Cloud аккаунт

### Установка
1. Клонируй репо
2. Установи зависимости (`npm install` или `composer install`)
3. Настрой конфиг базы данных
4. Запусти сервер
5. Открой http://localhost:3000 (или твой URL на Beget)
