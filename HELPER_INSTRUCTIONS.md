## ❓ Если что-то не работает

### **Ошибка: "mkdir: cannot create directory"**
→ Используй `sudo`: `sudo bash deploy_helper.sh`

### **Ошибка: "mysql: command not found"**
→ MySQL не установлен. Попроси хостинг установить или используй `mysql -u root < ...`

### **Ошибка: "Permission denied"**
→ Добавь `sudo` перед командой

### **Сайт не открывается в браузере**
→ Проверь что скрипт выполнился без ошибок
→ Проверь что файлы в правильной папке: `/home/elendes10/public_html/`
