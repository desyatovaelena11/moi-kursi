# Спецификация: Личная платформа видеокурсов

## Executive Summary

Веб-платформа для организации и просмотра личной коллекции видеокурсов (84GB, 20+ курсов). Подобна GetCourse, но для приватного использования одного пользователя. Видео хранится на облаке Mail.ru Cloud, интерфейс на Beget хостинге.

**Стоимость:** ~650 ₽/месяц (Beget 520₽ + Mail.ru Cloud 133₽)

---

## Проблема и цели

### Проблема
- Много видеокурсов записано со скрина (учебные материалы)
- Видео занимают 84GB на компьютере (неудобно)
- На платформах (GetCourse, autoweboffice) нет постоянного доступа (курсы закрываются)
- Видео разбросаны по разным местам - нет системной организации
- Нужен удобный доступ с разных устройств

### Цели
1. Создать единую платформу для всех видеокурсов
2. Систематизировать как на GetCourse (курсы → разделы → уроки)
3. Освободить место на компьютере (переместить на облако)
4. Удобный поиск и фильтрацию по курсам/темам
5. Доступ с браузера и мобильных устройств

### Успех: как это измерять?
- ✅ Все 84GB видео перемещены на Mail.ru Cloud
- ✅ Все видео доступны через веб-платформу и мобильно
- ✅ Быстрый поиск по названиям и фильтры работают
- ✅ Интерфейс удобен и интуитивен
- ✅ Видео воспроизводится плавно без лагов

### 📊 Структура данных

**Иерархия курсов:**
```
Платформа
└─ Курс 1 (название, описание, дата создания)
   ├─ Раздел 1
   │  ├─ Урок 1 (название, описание, ссылка на видео в облаке)
   │  ├─ Урок 2
   │  └─ Урок 3
   ├─ Раздел 2
   │  └─ Урок 4
   └─ Раздел 3
└─ Курс 2
   ├─ Раздел 1
   └─ Урок 1
```

**База данных содержит:**
- Курсы (name, description, created_at, order)
- Разделы (name, description, course_id, order)
- Уроки (name, description, section_id, video_url, duration, order)
- Теги (для поиска и фильтрации)

## Как это работает: Процесс использования

### 1️⃣ **Подготовка видео (разовая)**
```
Ты на компе → Видеофайлы (84GB)
   ↓
Открываешь Mail.ru Cloud
   ↓
Загружаешь видео в облако (структурируешь по папкам)
   ↓
Получаешь ссылки на видео
```

### 2️⃣ **Создание структуры на платформе**
```
Открываешь платформу (веб-интерфейс)
   ↓
"Добавить курс" → заполняешь название, описание
   ↓
"Добавить раздел" → структурируешь внутри курса
   ↓
"Добавить урок" → вставляешь ссылку на видео из облака
```

### 3️⃣ **Просмотр и использование**
```
Открываешь платформу в браузере / мобильном приложении
   ↓
Видишь список всех курсов
   ↓
Нажимаешь на курс → видишь разделы
   ↓
Нажимаешь на раздел → видишь уроки
   ↓
Нажимаешь на урок → видео воспроизводится из облака
   ↓
Можешь искать по названию или фильтровать
```

- Видео разбросаны по курсам и темам

### Шаг 2: Загрузка в Mail.ru Cloud (вручную)

- Вставляешь ссылки на видео из облака

## Риски и ограничения

### 📌 Основные риски

1. **Загрузка видео долгая**
   - 84GB в Mail.ru Cloud = недели загрузки
   - Риск: соединение обрывается
   - Решение: использовать приложение Mail.ru Cloud для десктопа (быстрее)

2. **Доступность из России**
   - Mail.ru Cloud полностью российский ✅
   - Beget доступен из России ✅
   - Но в будущем могут быть ограничения
   - Решение: готовить план B (миграция на другое облако)

3. **Скорость воспроизведения видео**
   - Если интернет медленный, видео может буферизироваться
   - Решение: автоматическое переключение качества видео

4. **Безопасность пароля**
   - Все твои видео в облаке — нужна сильная защита
   - Решение: двухфакторная аутентификация на Mail.ru
